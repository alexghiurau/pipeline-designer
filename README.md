# Pipeline Designer

An application for visualising data pipelines.

## Installation

Install pipeline-designer with [pnpm](https://pnpm.io/installation)

```bash
  pnpm install
```

Then start the development server

```bash
  pnpm run dev
```

## Features

- Visualise data pipelines
- Configure data pipelines
- Fullscreen mode
- Cross platform
